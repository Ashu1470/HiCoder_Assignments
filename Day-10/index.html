What is Meda Query


Media queries are a feature of CSS that enable webpage content to adapt to different screen sizes and resolutions. They are a fundamental part of responsive web design and are used to customize the appearance of websites for multiple devices.


b. What is media in CSS?

Media features describe specific characteristics of the user agent, output device, or environment. Media feature expressions test for their presence or value, and are entirely optional.


What is content management system?


A content management system or CMS is a software that is used to build websites and create content to be published on the internet. Typically CMS allows you to create a website without writing any code.
